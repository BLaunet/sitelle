Plot (`sitelle.plot`)
*********************************

.. currentmodule:: sitelle.plot

Introduction
============

This modules contains wrappers around :mod:`plt:matplotlib` methods to ease the plot of typical quantities like maps and spectra.
They all return a tuple (:class:`~plt:matplotlib.figure.Figure`, :class:`~plt:matplotlib.axes.Axes`), just like :func:`plt:matplotlib.pyplot.subplots`, to keep control over the axes.
