***********************************
Calibration (`sitelle.calibration`)
***********************************

.. currentmodule:: sitelle.calibration

Description
===========

This module gathers convenient functions that are used during the astrometric calibration of SITELLE data.
It defines an astrometric precision method, as well as a cross-matching algorithm.
Most of all, it provides wrapper to world2pix and pix2wolrd method using the third astrometric calibration defined by Martin et al. 2018
